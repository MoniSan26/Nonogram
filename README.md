# Nonogram Solver

Este proyecto consiste en una aplicación de modelado y programación para resolver **nonogramas** (rompecabezas lógicos basados en pistas numéricas).

Se implementan técnicas de lógica y programación para interpretar las restricciones de filas y columnas, generando soluciones que cumplen con las condiciones del problema.

El objetivo es aplicar herramientas de modelado computacional para resolver problemas de lógica de forma automática.
